About the API Edamam

The API Edamam provides a large recipe database with more than 2 million english recipes. They make sure to add new sites and recipes contiuously.

The API used in this project is the Recipe Search API. EDAMAM has other two APIs: Nutrition Analysis API and Food Database API.
